# 📊 Sales Analytics & Profitability Insights Dashboard

## 🚀 Project Overview
This project focuses on transforming raw retail data into **actionable business insights** using Power BI.  
The dashboard is designed for **CXO-level decision-making**, enabling leadership to monitor **sales performance, profitability, and regional trends** in real time.

It is part of a **Business Intelligence Transformation initiative** aimed at replacing static reports with interactive, data-driven dashboards.

---

## 🎯 Business Problem
A Mumbai-based electronic conglomerate generates large volumes of data but lacks **visual intelligence** to:

- Track real-time sales performance  
- Identify high and low-performing products  
- Monitor profitability and discount impact  
- Optimize inventory and regional strategies  

---

## 💡 Solution
Developed an **interactive Power BI dashboard** that:

- Consolidates sales and financial data  
- Provides **real-time KPIs and insights**  
- Enables **data-driven decision-making**  
- Highlights trends, patterns, and anomalies  

---

## 📌 Key Features

### 🔹 KPI Metrics
- Total Revenue  
- Gross Profit  
- Customer Count  
- Total Quantity Sold  
- Average Selling Price  

---

### 🔹 Core Visuals

| Area | Visualization | Purpose |
|------|-------------|--------|
| 📈 Performance | Gauge Chart | Track revenue vs target |
| 📊 Products | Bar Chart | Identify top-performing products |
| 📊 Categories | Column Chart | Analyze category contribution |
| 📉 Profitability | Scatter Plot | Understand discount impact |
| 🌍 Filters | Slicers | Drill-down by city, zone, division |

---

### 🔹 Visualizations Included

- 📈 **Revenue Performance vs Goal (Gauge Chart)**  
- 📊 **Product-wise Revenue Analysis (Bar Chart)**  
- 📊 **Category-wise Revenue Distribution**  
- 🗺️ **City & Regional Filters (Slicers)**  
- 📉 **Gross vs Discount Variance (Scatter Plot)**  

---

## 🧠 Key Insights

- 📌 A few product categories contribute the majority of revenue  
- 📌 Discounts have a direct impact on profitability  
- 📌 Certain regions outperform others in sales performance  
- 📌 Opportunity to optimize pricing and inventory strategies
